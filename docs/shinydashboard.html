8 shinydashboard

Shinydashboard is a package built on top of Shiny that provides a set of functions for creating dashboards. A dashboard is a type of web application that displays a set of metrics or KPIs (key performance indicators) in a visually appealing way. Shinydashboard provides a set of pre-designed UI elements, such as boxes, headers, and tabs, that can be combined to create a polished and professional-looking dashboard.

ui = fluidPage(
  tabsetPanel(
    tabPanel("Questionnaire", 
             fluidRow(
               br(),
              column(12, "PHQ-9", 
                     fluidRow(
                     column(5, "Questionnaire"), 
                     column(7, "Scores")
                     ))), 
             fluidRow(
              column(12, "",
               fluidRow(
                       column(5, "Little interest or pleasure in doing things"), 
                       column(6, radioButtons("q1", "",
                                              choiceNames = c('Not at all',
                                                              'Several',
                                                              'More than half',
                                                              'Nearly every'
                                              ), 
                                              choiceValues = c(0,1, 2, 3),
                                              inline = TRUE))
                     )
              ))
             ), 
    tabPanel("Results")
  )
)
